# WPF-Screen-Recording
A desktop screen recording application built using C# (.NET Framework 4.8) and WPF.  Features: • Full screen recording • Custom save location • MP4 output • Modern UI • Start / Stop controls  Tools: C#, WPF, Accord.FFMPEG
